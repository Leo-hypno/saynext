## Which File Should I Download?

| System | File |
| --- | --- |
| Mac Apple Silicon | `SayNext-0.1.3-darwin-aarch64-dmg.dmg` |
| Mac Intel | `SayNext-0.1.3-darwin-x64-dmg.dmg` |
| Windows | `SayNext-0.1.3-windows-x64-nsis-setup.exe` |

## Notes

- macOS builds are signed and notarized starting from `v0.1.3`.
- Windows builds are not code-signed yet, so Windows may show a SmartScreen warning.

- Users on `v0.1.0` should manually install `v0.1.1` once.
- Starting from `v0.1.1`, future versions can be installed through SayNext's in-app updater.
