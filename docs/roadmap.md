# Roadmap

## v0.1.0 Public MVP

- macOS and Windows desktop app
- Global hotkey
- Floating palette
- Context tabs
- Click or Enter to copy
- Native clipboard integration
- Tray/menu bar integration
- Tray/menu bar close behavior
- Native window controls and remembered position
- Minimal settings panel
- Favorites
- Recent prompt tab
- Custom prompts
- Manual appearance setting
- Traditional Chinese and English packs
- README, screenshots, and demo GIF
- GitHub launch checklist

## v0.2.0 Prompt Pack System

- Load packs from local folder
- Validate pack schema
- Import pack button
- Open packs folder
- Simplified Chinese pack

## v0.3.0 Customization

- Custom hotkey settings
- Add and edit prompts
- Export personal pack

## v1.0.0 Stable

- Stable cross-platform builds
- Auto update
- Accessibility pass
- Mature contribution workflow
