# SayNext Demo

This page collects the product visuals and demo flow used by the GitHub README.

## Core Flow

1. Open SayNext with `Cmd/Ctrl + Shift + H`.
2. Pick a context, such as starting a question, clarifying an answer, planning next steps, or checking the final result.
3. Copy a prompt with click or `Enter`.
4. Paste it into ChatGPT, Claude, Gemini, Codex, or another AI chat box.
