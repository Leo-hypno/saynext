# Design Principles

## Built for the Stuck Moment

SayNext is for the moment when someone does not know what to ask AI next.

## Short Beats Clever

Prompts should sound like something a real person would paste.

## Small and Quiet

The app should appear quickly, help, and get out of the way.

## Offline First

No account, no tracking, no AI API key.

## Community Friendly

Prompt packs should be easy to read, review, and contribute.
