# GitHub Repository Setup

These settings make SayNext easier to discover and easier to contribute to.

## Suggested Topics

Add these in the repository About panel:

```text
ai
prompt
desktop-app
tauri
productivity
macos
windows
beginner-friendly
open-source
```

## Suggested Labels

| Label | Use for |
| --- | --- |
| `good first issue` | Small, friendly starter tasks |
| `prompt-pack` | Prompt wording, scenarios, or language packs |
| `translation` | README, docs, UI, or prompt translations |
| `design` | Screenshots, icons, layout polish, demo GIFs |
| `windows` | Windows installer, tray, shortcut, or SmartScreen notes |
| `documentation` | README, guides, launch material, release docs |

## Good First Issue Ideas

- Add a Simplified Chinese prompt pack.
- Add a Japanese prompt pack.
- Improve English wording for everyday AI use cases.
- Record a short demo GIF for the README.
- Test the latest Windows installer and report any launch friction.
- Add screenshots for light mode and settings.
