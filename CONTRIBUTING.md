# Contributing to SayNext

Thanks for helping make AI easier to keep talking to.

SayNext accepts two kinds of contributions:

- Code contributions
- Prompt pack contributions

## Contribute Prompt Packs

Prompt packs are JSON files in `packs/`.

You can help by:

- Adding a new language
- Improving wording
- Adding missing beginner scenarios
- Making prompts shorter and more human

Before opening a pull request:

1. Keep prompts short.
2. Avoid jargon like "prompt engineering" unless the pack is for advanced users.
3. Write prompts people would actually paste into an AI chat.
4. Use stable ids in lowercase kebab-case.

## Contribute Code

The desktop app lives in `apps/desktop`.

Useful areas:

- Tauri global shortcut integration
- Clipboard integration
- Prompt pack loading
- Search quality
- Accessibility
- Cross-platform testing

## Pull Requests

Please include:

- What changed
- Why it helps
- Screenshots or GIFs for UI changes
- Manual test notes for desktop behavior
