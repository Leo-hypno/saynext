# SayNext Prompt Packs

Prompt packs are plain JSON files.

To contribute a pack:

1. Copy an existing pack.
2. Change `id`, `name`, `description`, and `locale`.
3. Keep category ids stable.
4. Add prompts that are short, direct, and useful when someone is stuck.

Good rescue prompts should help with one of these moments:

- I do not know how to start.
- I do not understand the answer.
- The answer is weak.
- The AI refused too early.
- I need research.
- I need a plan.
- I need execution.
- I need review.
